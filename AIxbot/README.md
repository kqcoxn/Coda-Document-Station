## 简介

### 它是什么？

AI 星宝是一个基于[Nonebot2](https://github.com/nonebot/nonebot2)框架的轻量化 QQ 机器人，作用于山理工校青协各个需要 QQ 群的工作场景。

得益于 Nonebot2 的强大特性，AI 星宝继承有以下特点：

- 易操作性
- 可扩展性
- 可操控性
- 无限制性
- ~~要修 bug~~（不是）

### 它能做什么？

在目前的框架中，它完全能够在纳新、赛事活动、志愿者招募群内胜任如下功能：

- 常见问题自动回复
- 诈骗信息提醒
- ~~水群~~（划掉）
- 任意需要的 QQ 内可以达成的功能
